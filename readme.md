An API Client of Red Dot Payment API v2. Can support below API:

* Payment API
    * Hosted Page
    * Redirection Hosted (Adding automation using puppeteer)
    * Direct API
* Tokenization
    * Redirection Page
    * Hosted page
* Merchant API
    * Void
    * Refund
    * Capture

## Reqiurement
* express
* moment
* body-parser
* js-sha512
* is_js
* underscore
* puppeter for the automation
